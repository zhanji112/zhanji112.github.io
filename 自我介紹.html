大家好，我是林昀蓁

個人興趣

我是資傳一甲的林昀蓁，今年19歲，在家中，我是年紀最小的，有一個比我大兩歲的哥哥。從小爸爸就在大陸工作，在生活上幾乎是由媽媽照顧我跟哥哥，在學習上支持我認真做自己喜歡的事，總是給予我很大的鼓勵和自由發展的空間， 也因此養成我獨立思考的態度與習慣。小時候，在寒暑假經常和家人出國旅遊，而日本是讓我印象最深刻的國家，因為旅行中處處有「遇見新事物」的驚喜， 從沿途裡的吉祥物、美食、科技、伴手禮⋯發現了「每個城市都各有特色」的日本文化，真的佩服他們結合廣告傳媒與行銷的功力，而其中的「動漫文化」是日本很特殊的產業鏈， 從中學習到該國的文化和藝術之外，同時啟發了我現在最熱衷的～cosplay，因為這個興趣，我學會了許多技能，像是基本的化妝，ps修圖、製作道具，還有知道如何宣傳自己，也對拍照、錄製影片產生了興趣。

高中社團

回顧我的求學過程是順利的，升上高中把大多心思花在社團方面，但卸下社團幹部職務後把重心漸漸移回課業上，在讀書方面盡我最大能力之外，我也會利用剩餘時間培養其他方面的能力與興趣。 上高中後，高一除了參加手語社外，也參加了衛生糾察，高二當了手語社的幹部。在班上，擔任了高二的班際啦啦隊負責人，高三負責班上畢業紀念冊的編輯工作。對我來說，只要是在自己能力之內做得到的事，可以踴躍參與的， 都是學習的好機會，還可以藉此增進自己的能力。在其他人眼中，對於我的第一印象可能是冷漠、不愛說話，但事實上，我是一個健談的人，也善與人溝通。在團體中，我可以擔任領導與溝通的角色，其中若有成員發生爭執時， 我會先釐清爭執點，然後再提出解決的方案，讓活動能順利的推展。而我也擅長接收各方面的資訊，將自己學到的事物融會貫通，相信這些能力是就讀傳播相關類型的科系非常需要的特質。 高一剛升進內湖高中，參加了學校所舉辦的社團博覽會，就深深地被手語社的學長姊帥氣的表演吸引住，便進入了這個社團。在手語社擔任了公關兼服裝造型，學到了公關要如何與外校溝通、代表社團與廠商接洽， 也學到了服裝造型要怎麼幫大家設計表演衣服、上台前幫大家化妝。我雖然幹部是公關兼服造，但我也會幫忙其他組別，如製作社服、成果發表介紹本、表演道具。只要有需要我的地方，我一定會盡我所能地給予幫助。 雖然我們這屆因為疫情關係，成發半途取消，我們只能改成用影片呈現，我也藉此研究了許多關於影片製作、剪輯的內容，碰巧看到了關於傳播系的介紹，讓我深感興趣，有了之後想要讀傳播相關科系的念頭，希望能更深入的學習。 回想參加社團的這些日子，社團真的是一個很特別的地方，可以學到書本上沒教的事物，待人處事、溝通能力和時間管理，我相信這是難能可貴的經驗。

喜好音樂團體

音樂方面我自己非常喜歡聽日本音樂，自己也很喜歡搖滾樂，其中搖滾樂團最喜歡的是MY FIRST STORY，會認識這個年輕樂團的原因很簡單，也跟大多數的人一樣。因為主唱HIRO跟ONE OK ROCK主唱TAKA的兄弟關係。 我會認識其他的JROCK樂團，幾乎都是因為ONE OK ROCK的關係。第一次在MY FIRST STORY官網上看他們的PV，不得不說HIRO真的跟TAKA有神似的地方。但也許是因為PV的拍攝風格，又加上兩團的音樂方向都差不多，才會有這種重疊的感覺。 up 因為仔細聽過MY FIRST STORY的音樂，會發現這兩團的風格還是有差別的。我不懂搖滾樂的類別怎麼分、JROCK這一塊領域支線很多，要深究其實並不容易。 這也是我會有現在這些想法的原因之一。隨著ONE OK ROCK的知名度越來越高，喜歡他們的人越來越多這一點開始，不只一次有人拿這兩個樂團來作比較、評論，甚至批評。我只覺得這對MY FIRST STORY來說，並不公平。 多少人只是因為聽過ONE OK ROCK就認為音樂路線相近的MY FIRST STORY是抄襲、是仿造，兩位主唱的外型有幾分神似那畢竟是因為血緣之親，唱歌的台風或手勢這一點，我真的想說那並不是刻意抄襲。 HIRO在廣播上曾分享過自己喜歡的樂團（このバンドでやりたい），除了coldrain之外，HIRO說了另外一個樂團クロスフェイス(Crossfaith)，上網看過那個樂團的演出之後， 我必須說，如果HIRO的台風是抄襲，我算不上是MY FIRST STORY的迷(以我自己的標準來說)，但是這個樂團的消息、音樂，還是有在關注。而且不得不說，我還真的滿喜歡MY FIRST STORY的音樂。他們的音樂很有層次性、耐聽也有味道。 我特別喜歡他們每一首歌長長的前奏。 我一定不了解每個團員的個性跟其他資訊，但是我想說，雖然這個樂團出道資歷還很年輕，但在音樂方面，聽得出來他們是玩真的。 我相信等HIRO在各方面都更成熟了之後(畢竟樂團主唱還是主要核心吧)，這個樂團要達到他們想望的目標(TOKYO DOME)，並非不可能的事情。我喜歡ONE OK ROCK，就是因為喜歡ONE OK ROCK，才不想要看見打著自己是ONE OK ROCK歌迷的名義， 做出一些難以理解，甚至是很沒禮貌的言行舉止的歌迷 HIRO跟TAKA就是親生兄弟，這一點永遠不會改變。選擇了自己有興趣的路線，正好哥哥也在那個世界裡面的HIRO，他就算是沒有想過會碰見什麼問題，現在也開始面對那些狀況了吧。 他有企圖心，而且還不小。至於這份企圖心將會帶他到哪裡、改變他什麼，那都是HIRO自己該去面對解決的事情了。他們該如何從中成長、學習，從組了樂團之後的所有事情中寫出自己的心境、找出更明確的樂團曲風， 才是他們的團名中STORY的意義吧。

喜好的女歌手

還有一個我非常喜歡的日本女歌手，aimer，Aimer的藝名取自法語的「愛」，，她的作風非常神秘，出身、年齡、本名皆未公開，早期甚至不露臉，就連演唱會也會透過燈光遮掩， 低調到令人好奇，直到2016年左右才開始將容貌公諸於世，她的長相與她的歌聲一樣，同樣有種空靈的氣質，又有點可愛，說是美女絕對不為過。在父親是樂團Bass手的家庭中長大，Aimer從小就接受音樂的薰陶， 學鋼琴、練吉他，也接觸爵士與藍調，並且非常喜歡唱歌，尤其喜歡模仿椎名林檎和宇多田光，但在中學時卻因為過度操練而導致聲帶損傷，醫生交代她盡量不要開口，更別說唱歌了，她幾乎不發一語長達半年之久。 因為曾經失去歌聲，也讓她更加確立想當歌手的夢想，在漸漸康復的過程中她也摸索出了自己的獨特嗓音，有點低沉、沙啞，卻充滿磁性。而至今尚未康復的舊傷也使她幾乎不會太過用力或激烈地唱歌， 讓她的歌聲有時候很輕柔、有點慵懶、還有空靈感，辨識度非常高，許多人一聽就著迷了。2011年，喜歡唱歌的女孩Aimer終於正式出道了，她的第一張單曲《六等星之夜》 就成了動畫《未來都市NO.6》的片尾曲， 也從此與動漫結下不解之緣，後來還獻唱了《死神》、《夏雪之約》、FATE系列等動畫。而她與操刀《進擊的巨人》配樂、在動漫界「超燃神曲製造機」之稱的作曲家澤野弘之合作時， 更是在收復歌聲後首度以「不那麼用力」的方式挑戰磅礡的搖滾曲風，一首《機動戰士鋼彈UC》主題曲《RE:I AM》讓她在抒情、空靈的曲風外有了新的面貌為人所知。最近她又搭讓了現象級動漫《鬼滅之刃》的熱潮獻唱主題曲《殘響散歌》， 人氣想必會再飛漲一波。自己非常喜歡日本的文化與生活，希望能在疫情結束後去日本玩，也想去聽聽看喜歡的樂團開的演唱會，以及去到許多心目中想去已久的景點與美食。 音樂方面我自己非常喜歡聽日本音樂，自己也很喜歡搖滾樂，以及去到許多心目中想去已久的景點與美食。
